git status时一直提示有未跟踪的文件:(new commits, untracked content)

进入子项目themes/next发现文件夹下有一个patch

删除就没问题了。

针对子项目中未跟踪的文件，可以分两种情况处理：

  1. 如果项目目中的文件是不需要提交的，则删除即可。
  2. 如果是需要提交的的，则先在子项目提交，再在主项目提交。
